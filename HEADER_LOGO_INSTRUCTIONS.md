# 🎯 Логотип в хедере формы записи

## Что изменено:

### ✅ **Логотип перемещен из футера в хедер**
- **Раньше:** Логотип был внизу в футере и в разделе работ
- **Теперь:** Логотип вверху в фиолетовом прямоугольнике рядом с названием компании

### ✅ **Структура хедера:**
```html
<div class="booking-header">
  <div class="business-info">
    <div class="business-logo-header">
      <div id="business-logo-header-container">
        <img id="business-logo-header" src="" alt="Logo">
        <div class="no-logo-header-text">Нет логотипа</div>
      </div>
    </div>
    <div class="business-details">
      <h1 class="business-name" id="business-name">Название компании</h1>
      <p class="business-meta" id="business-meta">Адрес, телефон</p>
    </div>
  </div>
</div>
```

### ✅ **Стили логотипа:**
- **Размер:** 60x60px (50x50px на мобильных)
- **Закругленные углы:** 12px
- **Позиция:** слева от названия компании
- **Отступ:** 16px между логотипом и названием

### ✅ **Мобильная версия:**
- **Вертикальное расположение:** логотип сверху, название снизу
- **Уменьшенный логотип:** 50x50px
- **Отступ:** 12px между элементами

## Как это выглядит:

### **Десктоп:**
```
[ЛОГОТИП 60x60]  Название компании
                 Адрес, телефон
```

### **Мобильный:**
```
[ЛОГОТИП 50x50]
Название компании
Адрес, телефон
```

## 🧪 Тестирование:

### 1. **Перезапустите сервер:**
```bash
pm2 restart ecosystem.config.js
```

### 2. **Проверьте в настройках:**
1. Настройки → Брендинг
2. Убедитесь что логотип загружен
3. Убедитесь что название компании введено

### 3. **Проверьте форму записи:**
1. Откройте страницу онлайн-записи
2. В фиолетовом прямоугольнике должно быть:
   - Логотип слева
   - Название компании справа
   - Адрес/телефон под названием

### 4. **Проверьте консоль:**
```javascript
// Должны быть логи:
"Logo data: /uploads/works/..."
"Company name set to: Ваша компания"
"Logo loaded successfully in header"
```

### 5. **Проверьте мобильную версию:**
1. Откройте DevTools (F12)
2. Переключитесь на мобильный вид
3. Логотип должен быть над названием

## 🔧 Если не работает:

### **Логотип не отображается:**
1. Проверьте console.log - есть ли "Logo data"
2. Проверьте API `/api/public/business?slug=...`
3. Проверьте что логотип загружен в настройках

### **Название не отображается:**
1. Проверьте console.log - есть ли "Company name set to"
2. Проверьте что название сохранено в настройках
3. Проверьте API `/api/business/name`

### **Стили съехали:**
1. Проверьте CSS классы `.business-info` и `.business-logo-header`
2. Проверьте что `display: flex` работает
3. Проверьте мобильные стили в `@media (max-width: 768px)`
