## 🔄 **Логика переходов:**

### **Автоматический переход:**
1. **Клик на опцию** → выделение + пауза 300мс
2. **Автоматически** открывается следующий шаг
3. **Кнопка "назад"** появляется после первого шага

### **История переходов:**
- Сохраняется история всех шагов
- Кнопка "назад" возвращает на предыдущий шаг
- Можно вернуться через несколько шагов

## 🧪 **Тестирование:**

### **1. Перезапустите сервер:**
```bash
pm2 restart ecosystem.config.js
```

### **2. Откройте форму записи:**
- Поле должно быть ужее (600px)
- Первый шаг должен загрузиться автоматически

### **3. Проверьте логику:**
- Клик на опцию → автоматический переход
- Кнопка "назад" → возврат на шаг
- Календарь → выбор даты → время

### **4. Проверьте адаптивность:**
- F12 → Mobile view
- Все элементы должны адаптироваться
