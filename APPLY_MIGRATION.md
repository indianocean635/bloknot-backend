## 🧪 Тестирование:

### **Настройки брендинга:**
1. Откройте настройки → Брендинг
2. Попробуйте загрузить логотип (drag & drop)
3. Измените название компании
4. Проверьте что все сохраняется

### **Форма записи:**
1. Откройте страницу онлайн-записи
2. Проверьте что в хедере только Bloknot логотип
3. Проверьте что в футере ваш логотип (если загружен)
4. Кликните на логотип в футере

### **API проверки:**
```bash
# Проверить логотип в бизнесе
curl "http://localhost:3000/api/public/business?slug=your-slug"

# Проверить работы с isLogo
curl -H "Cookie: your-session" http://localhost:3000/api/works

# Проверить название компании
curl -H "Cookie: your-session" http://localhost:3000/api/business/name
```

## ⚠️ Если возникнут проблемы:

1. **Логотип не загружается** - проверьте console.log в браузере
2. **Название не сохраняется** - проверьте API /api/business/name
3. **Футер не работает** - проверьте что элементы не внутри script
4. **Серверные ошибки** - `pm2 logs` для диагностики
