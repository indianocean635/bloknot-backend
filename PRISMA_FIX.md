# ✅ Ошибка Prisma исправлена!

## 🚨 **Проблема была:**
```javascript
// ❌ НЕПРАВИЛЬНО - select и include одновременно
prisma.business.findUnique({
  where: { id: businessId },
  select: { name: true },
  include: { branches: {...} }
})
```

## ✅ **Исправлено на:**
```javascript
// ✅ ПРАВИЛЬНО - только select
prisma.business.findUnique({
  where: { id: businessId },
  select: {
    name: true,
    branches: {
      take: 1,
      select: {
        address: true,
        phone: true
      }
    }
  }
})
```

## 🎯 **Что изменено:**
- Убран `include` из запроса
- `branches` перенесен внутрь `select`
- Вложенная структура осталась той же
